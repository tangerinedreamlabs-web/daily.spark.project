# Welcome Template Assets

Upload these files to your web host, then replace the matching URLs in `welcome.html`.

| Local file | Replace this URL in HTML |
| --- | --- |
| `welcome-header.png` | `https://your-domain.example/daily-spark/welcome/welcome-header.png` |
| `welcome-support-app.png` | `https://your-domain.example/daily-spark/welcome/welcome-support-app.png` |
| `welcome-benefits.png` | `https://your-domain.example/daily-spark/welcome/welcome-benefits.png` |
| `welcome-closing.png` | `https://your-domain.example/daily-spark/welcome/welcome-closing.png` |
| `welcome-cta-band.png` | `https://your-domain.example/daily-spark/welcome/welcome-cta-band.png` |

SendGrid email images must use absolute `https://` URLs. After upload, use the final public image URLs in the HTML before importing the template.
