# Mobile-first single-page website starter

This is a plain HTML/CSS/JS starter site. It is designed to be easy to customize and deploy.

## Files

- `index.html` — page structure and content
- `styles.css` — mobile-first responsive styling
- `script.js` — mobile nav and current year
- `netlify.toml` — optional Netlify config

## How to preview locally

Open `index.html` in your browser.

For a better local server:

```bash
python3 -m http.server 8080
```

Then visit:

```text
http://localhost:8080
```

## How to deploy on Netlify

1. Go to Netlify.
2. Create a new site.
3. Drag this folder into the deploy area.
4. Publish.
5. Optional: connect your custom domain.

## How to deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload these files.
3. Go to Settings → Pages.
4. Choose the main branch as the source.
5. Publish.
